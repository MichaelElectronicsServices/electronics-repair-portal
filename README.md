 #  Electronics Repair Portal
 
 A Django-based web portal for managing repair tickets, customer interactions, and technician workflows.
 
 ##  Features
 
 - Customer ticket submission with item info and contact
 - Technicians can add updates, estimates, schedules
 - MySQL backend for robust data tracking
 - Dockerized setup for quick local deployment
 
 ---
 
 ##  Tech Stack
 
 - Python 3.11 / Django 4.2+
 - MySQL 8.0
 - Docker + docker-compose
 - HTML templates with Django templating
